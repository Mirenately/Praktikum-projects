# Praktikum-projects/Afisha_sales_analysis

**Проект "Исследование сервиса Яндекс.Афиша"**

**Описание проекта**: Исследование поведения пользователей сервиса Яндекс.Афиша: анализ посещений сервиса и совершаемых покупок. Расчет эффективности маркетинговых инвестиций для определения наиболее перспективных направлений.

**Исследуемые данные**: Логи посещений сервиса, данные о покупках, информация о маркетинговых инвестициях.

**Используемые инструменты**: Исследовательский анализ, статистический анализ, когортный анализ бизнес-показателей (Retention rate, conversion rate, воронка продаж), анализ маркетинговых инвестиций (ROI, Lifetime value - LTV)

**Используемые библиотеки**: pandas, matplotlib, scipy, math, seaborn
