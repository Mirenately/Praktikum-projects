# Praktikum-projects/Telecom_analysis

**Проект "Исследование тарифных планов мобильного оператора Мегалайн"**

**Описание проекта**: Проведение сравнительного анализа тарифов мобильного оператора для определения наиболее прибыльного для компании. Выявление сильных и слабых сторон тарифов.

**Исследуемые данные**: Информация о совершённых звонках, отправленных сообщениях, интернет-траффике, база данных пользователей.

**Используемые инструменты**: Исследовательский анализ, статистический анализ

**Используемые библиотеки**: pandas, matplotlib, scipy, math
