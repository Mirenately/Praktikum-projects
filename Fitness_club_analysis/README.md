# Praktikum-projects/Fitness_club_analysis
**Проект "Исследование поведения клиентов фитнес-центра"**

**Описание проекта**: Исследование поведения клиентов сети фитнес-центров. Изучение параметров оттока, кластеризация клиентов и выделение наиболее перспективных групп.

**Исследуемые данные**: База данных по посещениям фитнес-центров (дата первого посещения, средняя частота посещений за период, факт оттока, участие в промо-акциях)

**Используемые инструменты**: Исследовательский анализ, статистический анализ, регрессионный анализ

**Используемые библиотеки**: pandas, matplotlib, scipy, math, seaborn, plotly, sklearn
