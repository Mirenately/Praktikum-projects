# Praktikum-projects/Gaming_market

**Проект "Исследование рынка видеоигр"**

**Описание проекта**: Исследование рынка видеоигр для определения перспективных платформ и жанров, а также выявление особенностей региональных рынков, влияющих на структуру продаж.

**Исследуемые данные**: База данных о реализации видеоигр с 1980 по 2016гг. с информацией о жанрах, платформе, продажах в разных регионов, оценках критиков и пользователей, возрастном рейтинге.

**Используемые инструменты**: Исследовательский анализ, статистический анализ

**Используемые библиотеки**: pandas, matplotlib, scipy, math
