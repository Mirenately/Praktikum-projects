# Praktikum-projects/Online-shop_ab-testing

**Проект "Исследование интернет-магазина"**

**Описание проекта**: Исследование гипотез по повышению продаж интернет-магазина, изучение результатов проведённого A/B-тестирования.

**Исследуемые данные**: Данные о посещениях интернет-магазина, информация о продажах, характеристика гипотез по повышению продаж.

**Используемые инструменты**: Исследовательский анализ, A/B-тестирование, приоритизация гипотез (методы ICE, RICE)

**Используемые библиотеки**: pandas, matplotlib, scipy, math, seaborn
